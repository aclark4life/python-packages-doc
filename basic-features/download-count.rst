Display download count
----------------------

In its simplest form, `pythonpackages.com`_ allows signed-in users to enter a Python package name

.. image:: https://github.com/pythonpackages/pythonpackages-docs/raw/master/basic-features/download-count1.png
   :class: thumbnail

to see the number of times it has been downloaded from the `Python Package Index`_:

.. image:: https://github.com/pythonpackages/pythonpackages-docs/raw/master/basic-features/download-count2.png
   :class: thumbnail

.. _`pythonpackages.com`: http://pythonpackages.com
.. _`Python Package Index`: http://pypi.python.org/pypi

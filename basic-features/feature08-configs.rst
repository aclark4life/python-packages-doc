Advanced
========

In this section we describe an advanced feature of `pythonpackages.com`_: hosted `zc.buildout`_ configuration files.

Hosted configuration files
--------------------------

In addition to its core features, ``pythonpackages.com`` provides a set of shared, extendable configuration files for building Python-based and other software.  Please continue reading to see examples.

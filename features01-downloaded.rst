Display number of downloads
---------------------------

In its simplest form, `pythonpackages.com`_ allows you to enter a Python
package name

.. image:: https://github.com/aclark4life/pythonpackages-docs/raw/master/features01-downloaded-00.png

to see the number of times it has been downloaded from the `Python Package
Index`_:

.. image:: https://github.com/aclark4life/pythonpackages-docs/raw/master/features01-downloaded-01.png

Exciting!

.. _`pythonpackages.com`: http://pythonpackages.com
.. _`Python Package Index`: http://pypi.python.org/pypi
